# Selenium Automation 

### Automatin Project Created With
- Selenium WebDriver
- xUnit
- C#
- .Net
- Visual Studio IDE
- Allure


## Setup
1. Clone repository ``https://github.com/kvale17/SeleniumAutomationCS.git``
2. Restore project ``dotnet restore``

## Run Tests
``dotnet test``

## Generate and Open Allure Report
``allure serve``
